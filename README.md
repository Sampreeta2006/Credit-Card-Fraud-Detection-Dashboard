# Credit Card Fraud Analytics Dashboard
## Project Overview

This project analyzes credit card transactions to identify fraudulent activities using Python and Power BI. The dashboard provides insights into fraud patterns, transaction distribution, transaction amounts, and risk categories.

## Tools & Technologies

* Python
* Pandas
* NumPy
* Power BI

## Dataset

* Credit Card Fraud Detection Dataset
* Total Transactions Analyzed: 284,807
* Fraud Transactions: 473

## Project Features

### Executive Summary Dashboard

* Total Transactions Analysis
* Fraud Transactions Analysis
* Normal Transactions Analysis
* Fraud Rate Calculation
* Fraud vs Normal Transactions Visualization
* Transaction Distribution Analysis
* Transaction Amount Analysis

### Fraud Risk Analysis Dashboard

* Risk Category Classification
* Fraud Risk Analysis
* Fraud Transaction Investigation Table
* Risk-Based Insights

## Key Insights

* Total Transactions Analyzed: 284,807
* Fraud Transactions Detected: 473
* Fraud Rate: 0.17%
* Legitimate Transactions Account for 99.83% of All Transactions
* Medium and High-Risk Transactions Contribute Significantly to Fraudulent Activity
* Risk Categorization Helps Identify Potentially Suspicious Transactions for Further Investigation
