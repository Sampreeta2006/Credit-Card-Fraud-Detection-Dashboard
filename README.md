# Credit Card Fraud Analytics Dashboard
## Project Overview
This project analyzes credit card transactions to identify fraudulent activities using Python and Power BI. The dashboard provides insights into transaction patterns, fraud rates, and transaction amount distributions.

## Tools Used
- Python
- Pandas
- NumPy
- Power BI

## Dataset
- Credit Card Fraud Detection Dataset
- Total Transactions Analyzed: 284,807

## Features
- Fraud Transaction Analysis
- Fraud Rate Calculation
- Transaction Distribution Visualization
- Transaction Amount Analysis
- Interactive Power BI Dashboard

## Key Insights
- Total Transactions: 284,807
- Fraud Transactions: 473
- Fraud Rate: 0.17%
- Legitimate transactions account for 99.83% of all transactions.
- Fraudulent transactions represent a very small percentage but pose significant financial risk.
